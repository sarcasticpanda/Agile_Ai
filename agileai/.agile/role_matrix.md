# Role Matrix - @RoleAuth

| Role | Permissions |
|------|-------------|
| **ADMIN** | all routes, all UI, user management, system stats |
| **PM** | create/edit/delete projects & sprints, assign tasks, view analytics, view AI insights, CANNOT manage users |
| **DEVELOPER** | update OWN task status only, log work, add comments, view board & analytics, CANNOT create projects or sprints |
