# Agile AI

A repo for prototyping agile models integrated with AI.

## Quick Start Setup

To run both the backend and frontend, follow these simple steps:

### 1. Start the Backend
Open a terminal and navigate to the server folder:
``powershell
cd agileai/server
npm install
node server.js
``

### 2. Start the Frontend
Open a second terminal and navigate to the client folder:
``powershell
cd agileai/client
npm install
npm run dev
``
